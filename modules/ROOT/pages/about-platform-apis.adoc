Anypoint Exchange provides API features for publishing
and consuming assets by using the Exchange API, publishing
and consuming assets by using the Exchange Maven Facade API, and searching
for assets by using the Exchange Graph API.

The Maven facade also enables you to publish federated assets.

* https://anypoint.mulesoft.com/exchange/portals/anypoint-platform/f1e97bc6-315a-4490-82a7-23abe036327a.anypoint-platform/exchange-experience-api/[Exchange API]
